Project Overview
This repository serves as my Digital Literacy Portfolio, a collection of resources and reflections designed to help my peers navigate the digital world safely and professionally. This project covers five key tasks ranging from Basic Knowledge,professional branding to cybersecurity.

Task 1: Digital Literacy Infographic

Created a visual infographic using tools like Canva to explain key concepts of digital literacy such as safe internet usage, digital tools, and professional online presence.

Task 2: Student Digital Portfolio

Set up and updated professional profiles on platforms like GitHub and LinkedIn. This task helped in building an online presence and understanding how these platforms are useful for future career opportunities.

Task 3: Coding & Collaboration Platforms
Completed a beginner-level coding challenge on a platform like HackerRank.
Created a Google Form quiz on digital literacy awareness and analyzed responses.

Task 4: Email Etiquette & Communication
Drafted two professional emails (to a professor and internship coordinator).
Created a checklist of Do’s and Don’ts for responsible social media usage.

Task 5: Cybercrime Awareness
Developed a case study on a cybercrime topic (e.g., phishing or cyberbullying).
Created a prevention checklist with practical tips for staying safe online.
